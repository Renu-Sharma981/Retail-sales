# Retail-sales
🚀 Turning Raw Data into Business Insights!

I recently completed a Sales Performance Analysis Project (2022–2024) using Microsoft Excel, where I transformed raw transactional data into an interactive dashboard.

🔍 What I did:
✔️ Cleaned and structured data using Power Query
✔️ Created new columns in Power Query: Year, Month, Month Name & Revenue
✔️ Built dynamic PivotTables & charts
✔️ Designed a KPI-driven dashboard (Revenue, Units Sold, Growth %)
✔️ Analyzed performance by Category, Brand, Region & Channel

📊 Key Insights:
📈 Revenue grew by ~164% over 3 years
🥇 Yogurt category emerged as the top performer
🛒 Balanced contribution from Retail, E-commerce & Discount channels
⚠️ Promotions showed limited impact → needs better strategy

💡 What I learned:

Data cleaning is the most important step
Power Query makes data transformation faster & efficient
Dashboards help in better decision-making
